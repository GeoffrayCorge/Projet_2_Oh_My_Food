pwd
affiche le nom et chemin du répertoire courant. Elle permet de savoir où on se situe sur notre machine ;

ls
affiche la liste des fichiers et répertories dans un répertoire ;

cd
permet de changer de répertoire, c’est-à-dire de se déplacer d’un répertoire à un autre ;

mkdir
permet de créer un nouveau répertoire (ou “dossier”) ;

touch
permet de créer un fichier ;

mv
permet de renommer ou de déplacer des fichiers.
